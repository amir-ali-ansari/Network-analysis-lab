# Network-analysis-lab

## Overview
This repository documents my hands-on cybersecurity labs focused on network traffic analysis, packet inspection, and identifying suspicious activity.

I am building this portfolio as a cybersecurity student to demonstrate practical skills in tools such as Wireshark, TCP/IP analysis, DNS investigation, and security documentation.

## Skills Demonstrated
- Network traffic analysis
- Packet inspection
- TCP/IP fundamentals
- DNS and HTTP traffic review
- Suspicious activity identification
- Technical documentation

## Tools Used
- Wireshark
- Linux command line
- Windows networking tools
- Nmap

## Labs

### Lab 1: Wireshark Traffic Analysis
Analyze captured network packets to identify protocols, source and destination IP addresses, and unusual traffic patterns.

### Lab 2: how to read .pcsp file 
Review the traffic captured in another computer or another time.


## Career Connection
These labs support my goal to become a cybersecurity analyst by building real-world skills in monitoring, investigation, documentation, and threat detection.
